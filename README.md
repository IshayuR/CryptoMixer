# CryptoMixer: DEX Trading Prediction

This repository contains the code, dataset artifacts, figures, and experiment outputs for a course project studying a CryptoMixer-style architecture for fine-grained decentralized exchange (DEX) trading prediction on the Uniswap V2 USDC-ETH pool.

## Overview

The project centers on a lightweight MLP-based architecture for predicting user trading activity from sparse market observations. The implementation includes:

- a CryptoMixer-inspired PyTorch model with market augmentation and fusion components
- recurrent baselines for comparison
- a data pipeline for Uniswap V2 swap extraction and processing
- saved plots and metrics used in the accompanying report

## Repository Structure

- `models.py`: CryptoMixer-style model and LSTM/GRU baselines
- `train_cryptomixer.py`: training, evaluation, benchmarking, and ablation workflow
- `crypto_data_pipeline.py`: Uniswap V2 data processing and exploratory figure generation
- `data/`: canonical dataset CSVs used for inspection and grading
- `docs/augmentation.md`: concise explanation of the augmentation strategy used in the project
- `results/`: saved metrics tables and figures generated by the experiment workflow

## Method Summary

The modeling pipeline is organized around three ideas:

- sparse market observations are enriched with an attention-based market augmentation block
- user history and market context are mixed before prediction
- temporal and user-conditioned representations are fused for binary trade prediction

For a focused explanation of how augmentation is handled in the codebase, see [docs/augmentation.md](/Users/ishayuray/CSE 5819/CryptoMixer/docs/augmentation.md:1).

## Reproducibility

Dependencies are listed in `requirements.txt`.

Typical entry points:

```bash
python3 train_cryptomixer.py
python3 crypto_data_pipeline.py
```

The data pipeline supports live querying from The Graph when a `GRAPH_API_KEY` environment variable is available. The canonical CSVs included in this repository should be treated as the reference dataset snapshot for this project.
